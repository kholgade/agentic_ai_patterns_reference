# Multi-agent Execution Model

## Description
Coordination of multiple agents with defined roles and communication patterns.

## Used by
- Supervisor Pattern → [supervisor-pattern](../../patterns/supervisor-pattern/README.md)
- Hierarchical Team → [hierarchical-team](../../patterns/hierarchical-team/README.md)
- Agent Swarm → [agent-swarm](../../patterns/agent-swarm/README.md)
- Debate Pattern → [debate-pattern](../../patterns/debate-pattern/README.md)

## Properties
- distributed
- role-based
- emergent behavior possible

## When NOT to use
- simple sequential tasks
- tasks requiring strict deterministic output
